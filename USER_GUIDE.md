# 个人报销系统：朋友使用说明

这是一套在你自己 Mac 上运行的个人报销小工具。它可以读取微信、支付宝、银行卡和信用卡导出的 `.xlsx`、`.xls`、`.csv` 账单，帮助你筛选公司消费并导出报销表。

## 1. 第一次使用前

先安装 Node.js：

- 官网：https://nodejs.org/
- 建议安装 Node.js 22.12 或更新的 LTS 版本。
- 安装后重新打开“终端”，输入 `node -v` 和 `npm -v`，能看到版本号即可。

压缩包必须先完整解压。不要直接在压缩包预览窗口中双击文件。

## 2. 启动工具

### 方法 A：双击启动

1. 打开解压后的文件夹。
2. 双击 `start.command`。
3. 第一次启动会自动安装依赖，可能需要几分钟。
4. 保持弹出的终端窗口打开。
5. 在浏览器访问 `http://127.0.0.1:5173/`。

如果 macOS 提示脚本“已损坏”或无法打开，请阅读同一文件夹里的 `IF_CANNOT_OPEN.md`。

### 方法 B：在终端启动

打开“终端”，输入 `cd `（后面留一个空格），把解压后的文件夹拖进终端并回车，然后执行：

```bash
npm ci
npm run dev
```

浏览器访问：

```text
http://127.0.0.1:5173/
```

## 3. 三步完成报销

### 第一步：上传账单

- 进入“上传账单”页。
- 上传微信、支付宝、银行卡或信用卡导出的账单，可以一次选择多个文件。
- 第一次体验可以使用 `sample-data/` 文件夹里的三份模拟账单。
- 如有以前导出的报销表，也可以使用“上传以前的报销结果”恢复勾选和编辑内容。

### 第二步：筛选公司消费

- 在“消费筛选”页使用表头筛选、搜索、金额排序或隐藏列查找记录。
- 勾选真正需要报销的公司消费。
- 填写或修改报销月份、报销人、项目、费用类别和备注。
- 大额消费、携程订单、工作日餐饮和高速费规则只负责辅助筛入，你仍可取消勾选或修改分类。

### 第三步：检查并导出

- 进入“报销结果”页做最后检查。
- 如需使用“节假日前后高速”筛除规则，可以在操作后复原最近一次对应筛除。
- 点击导出，选择 Excel 或 CSV。
- 导出文件会保存到浏览器设置的下载位置。

## 4. 保存与恢复

- “保存本地进度”只保存在当前 Mac 的当前浏览器里，不会自动同步到其他电脑或浏览器。
- 这个分享包不包含 Musk 的历史报销结果，所以“一键恢复上一次报销结果”默认可能不可用。
- 你可以上传自己以前导出的 Excel 或 CSV 报销表恢复历史结果。

## 5. 飞书同步，可选

不使用飞书时可以完全跳过，不影响上传、筛选和导出。

如需同步到你自己的飞书多维表格，建议让懂技术的人或模型按照 `START_HERE_AI.md` 配置。必须使用你自己的飞书应用和多维表格权限，不要使用 Musk 的凭据，也不要把 `.env.local` 发给别人。

## 6. 停止工具

回到正在运行工具的终端窗口，按 `Control + C`；或直接关闭该终端窗口。浏览器页面随后无法继续使用，重新启动即可恢复访问。

## 7. 隐私说明

- 分享包不包含 Musk 的真实账单、历史报销结果、验证截图或飞书凭据。
- 你上传的账单在自己的浏览器和本机服务中处理。
- 本地保存的进度不会自动上传到 Musk 的电脑。
- 你主动导出文件或配置飞书同步后，会分别产生本地下载文件或远端飞书记录，请自行确认内容和接收范围。

## 8. 常见问题

### 页面打不开

确认运行工具的终端窗口没有关闭。如果 `http://127.0.0.1:5173/` 仍打不开，先停止旧终端中的工具，再重新双击 `start.command`。

### 启动文件显示“已损坏”

按照 `IF_CANNOT_OPEN.md` 操作。这通常是 macOS 给聊天软件或浏览器下载的脚本加了隔离标记，不代表源码损坏。

### 依赖安装失败

确认网络可用、Node.js 版本符合要求，并把终端中的完整错误信息发给协助部署的人或模型，不要只发“打不开”。

### 飞书同步失败

常见原因是本机飞书配置不完整、应用没有多维表格权限，或目标表没有把应用加入协作者。上传、筛选和导出不依赖飞书，可以先继续使用这些功能。
